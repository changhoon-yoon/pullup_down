풀업 · 풀다운 저항의 물리적 전류 흐름

스위치를 눌러서 전자가 어디로 흐르는지 직접 눈으로 확인하세요

풀업 (Pull-Up) 저항

저항이 +5V에 연결 → 평소엔 HIGH
열림 → MCU = HIGH (5V)
초록 = 도선주황 = 부품노랑 = 전류(전자 흐름)

스위치 열림: 전류는 저항을 지나 MCU 입력 핀을 5V로 살짝 끌어올림 (거의 안 흐름).
스위치 닫힘: 전류가 저항 → 스위치 → GND로 쏟아져 내려가고, MCU 핀은 0V가 됨.

풀다운 (Pull-Down) 저항

저항이 GND에 연결 → 평소엔 LOW
열림 → MCU = LOW (0V)
초록 = 도선주황 = 부품노랑 = 전류(전자 흐름)

스위치 열림: 위쪽이 막혀 전류가 못 내려옴. 저항이 MCU 핀을 0V로 끌어내림.
스위치 닫힘: +5V → 스위치 → MCU 핀 → 저항 → GND로 흐름. MCU 핀은 5V.

🧠 물리 기초 — 왜 이렇게 되는지 차근차근

1. 전압은 "높이", 전류는 "물의 흐름"

전기 회로를 이해하는 가장 쉬운 비유는 물입니다.

💧 수도관 비유
높은 물탱크 → 수도관 → 낮은 배수구
물은 높은 곳에서 낮은 곳으로 흐릅니다.
⚡ 전기 회로
+5V (높음) → 도선 → GND 0V (낮음)
전류는 전압이 높은 곳에서 낮은 곳으로 흐릅니다.
  • 전압(V) = 두 지점 사이의 "높이 차이". 단위는 볼트.
  • 전류(I) = 실제로 흐르는 전자의 양. 단위는 암페어(A).
  • 저항(R) = 흐름을 방해하는 좁은 관. 단위는 옴(Ω).

2. 옴의 법칙: V = I × R

이 세 가지는 공식 하나로 묶입니다. V = I × R — 전압은 전류 곱하기 저항.
수도관 비유로: 높이 차가 클수록(V↑) 물이 많이 흐르고(I↑), 관이 좁을수록(R↑) 덜 흐릅니다.

3. "떠 있는 핀(floating)" 문제 — 왜 저항이 필요한가

MCU(마이크로컨트롤러)의 입력 핀은 매우 민감한 안테나와 같습니다. 아무 것에도 연결되지 않으면 공기 중 전자기파를 잡아서 HIGH도 LOW도 아닌 애매한 상태가 됩니다. 버튼이 안 눌렸는데 눌린 것처럼 오동작하죠.

그래서 평소 상태를 확실히 정해 주는 저항이 필요합니다:

  • 풀업 저항: 평소 핀을 5V 쪽으로 당겨둠 → 버튼 누르면 0V
  • 풀다운 저항: 평소 핀을 0V 쪽으로 당겨둠 → 버튼 누르면 5V

4. 왜 저항값이 10kΩ쯤인가? (에너지 낭비)

저항이 너무 작으면(예: 100Ω): 버튼을 누를 때 전류가 5V / 100Ω = 50mA나 흘러 전력 낭비.
저항이 너무 크면(예: 10MΩ): 흐름이 거의 없어서 잡음을 못 이김.
10kΩ은 황금 비율 — 5V / 10kΩ = 0.5mA로 낭비 적고 잡음에도 강합니다.

5. 전자는 사실 반대 방향으로 흐른다 (재미있는 진실)

교과서에서 "전류는 +에서 −로 흐른다"고 합니다. 이건 약속입니다. 실제 전자(−전하)는 반대로 −에서 +로 움직여요. 벤자민 프랭클린이 전자를 알기 전에 방향을 정해버려서 생긴 역사적 실수인데, 이제 와서 바꾸면 혼란스러워 그냥 씁니다. 회로 분석에는 "전류 방향"만 쓰면 충분합니다.

6. 다음에 공부하면 좋은 것들

  • 키르히호프 법칙 — 갈림길에서 전류가 어떻게 나뉘는지 (물 비유로 바로 이해됨)
  • 커패시터 — 전기를 잠깐 저장하는 "물탱크"
  • 디지털 HIGH/LOW 임계값 — MCU는 대략 2V 이상이면 HIGH, 0.8V 이하면 LOW로 인식
  • 풀업 내장 — 대부분의 MCU(아두이노 포함)는 내부에 풀업 저항이 있어서 INPUT_PULLUP 한 줄로 켤 수 있음
